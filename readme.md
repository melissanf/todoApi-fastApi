# todo api 

## 1. project goal 
* create a backend api that allows users to manage tasks 
    - create tasks 
    - get all tasks 
    - get task by id 
    - upate task 
    - delete task 
* additional features 
    - mark task as completed
    - filter tasks by status 
    - pagination 
    - sort tasks by date 
## 2. stack used : 
- FastApi
- SQLAlchemy
- SQLite
- Pydantic
